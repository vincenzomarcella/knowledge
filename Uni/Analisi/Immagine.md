# Immagine
L'immagine di una funzione Ã¨ l'insieme di valori che una funzione assume nel suo [[Dominio|dominio]]. Infatti data una funzione $f:D\subseteq\mathbb{R}\rightarrow\mathbb{R}$ con [[Dominio|dominio]] $D=Dom(f)\subseteq\mathbb{R}$, viene detto immagine di $f$, l'insieme: 
$$Im(f)=\{y\in\mathbb{R} : y=f(x) \ \forall x \in D\}$$
